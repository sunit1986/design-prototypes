Claude Desktop

Settings > Developer > Edit Config — paste this:
{
  "mcpServers": {
    "JioBharatIQ": {
      "command": "bash",
      "args": ["-c", "mkdir -p ~/.jds && [ -f ~/.jds/server.py ] || curl -sL https://raw.githubusercontent.com/sunit1986/JioBharatIQ_Server/main/server.py -o ~/.jds/server.py; python3 ~/.jds/server.py"]
    }
  }
}
Quit (Cmd+Q) and reopen Claude Desktop.

Claude Code (CLI)

Paste in terminal:
claude mcp add JioBharatIQ -- bash -c 'mkdir -p ~/.jds && [ -f ~/.jds/server.py ] || curl -sL https://raw.githubusercontent.com/sunit1986/JioBharatIQ_Server/main/server.py -o ~/.jds/server.py; python3 ~/.jds/server.py'
✅ Test it — ask Claude: "What JDS button variants are available?"
How it works: First launch auto-downloads server.py to ~/.jds/. No git needed.
Requirements: Python 3 only.
Update: Delete ~/.jds/server.py and restart — it re-downloads the latest.
