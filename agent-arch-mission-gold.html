Request Flow
1
Mission Ingestion
User submits a natural-language mission. Context Injector enriches it with session history and retrieved memories from the Vector Store.
2
Goal Decomposition
Goal Decomposer breaks the mission into a directed acyclic graph of sub-goals. Strategy Planner assigns execution modes: tool-call, LLM-generate, or delegate to sub-agent.
3
Parallel Execution
Tool Router dispatches independent tasks concurrently. Sub-Agent Spawner handles long-horizon tasks requiring dedicated context windows. All results are written to Mission State.
4
Validation Loop
Each result is scored by Confidence Scorer. Low-confidence outputs trigger a re-plan cycle back to the Strategy Planner. Guardrails block policy-violating actions before emission.
5
Output Synthesis
Response Synthesizer merges all sub-results into a coherent, citation-backed answer. Action Emitter fires committed side-effects. Status Reporter streams completion events to the client.
